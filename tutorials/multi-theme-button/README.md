# Building a React Native Multi-Theme Button

## Getting Started

```
$ git clone https://github.com/HandlebarLabs/react-native-examples-and-tutorials.git
$ cd react-native-examples-and-tutorials/tutorials/multi-theme-button
$ yarn install
$ yarn run ios
$ yarn run android
```
